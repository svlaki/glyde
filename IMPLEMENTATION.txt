# Glyde - Personal Intelligence Operating System

## 📐 **Architecture Decisions**

### **Archetype → Category Migration** (Completed December 2024)
- **Decision**: Replaced complex archetype system with simple categories
- **Rationale**: Archetypes were too rigid, categories are flexible and user-defined
- **Impact**: Removed 342+ lines of frontend code, simplified backend logic
- **Benefits**: Better UX, easier to extend, AI can learn user-specific categories

### **Design System Implementation** (January 2025)
- **Decision**: Created centralized design tokens and enhanced Tailwind config
- **Rationale**: Needed consistent styling across entire application
- **Impact**: Foundation for modern UI polish
- **Benefits**: Maintainable theming, dark mode support, smooth animations

### **Calendar Page Layout** (January 2025)
- **Decision**: Split right panel into chat (top) + task list (bottom)
- **Rationale**: User requested better organization, task visibility
- **Impact**: Improved task awareness while maintaining AI assistant access
- **Benefits**: Better space utilization, task prioritization visible at glance
